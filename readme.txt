# 🌤️ MET.CO Weather Cards for Home Assistant
*A clean and visual way to display MET.CO forecast data inside Home Assistant*

This repository presents my custom setup for showing weather data from the **MET.CO integration** in Home Assistant.  
The project includes three purpose-built cards that make both short-term and long-term forecasts easy to read at a glance.

---

## ✨ Features

- **5-Day Forecast Card** – A compact overview of the next five days with temperatures, wind, and precipitation.  
- **5-Hour Forecast Card** – A detailed hour-by-hour snapshot of the upcoming five hours.  
- **Combined Forecast Card** – A single card that merges both views with a toggle switch for quick switching.

## 🧩 How It Works

All cards are powered by:

- The **MET.CO integration** in Home Assistant  
- **Daily** and **hourly** forecast entities  
- A layout optimized for **Sections View**  
- A clean structure that highlights the most important weather details

## 🚀 Purpose

This repository serves as a practical example of how MET.CO forecast data can be displayed in a **clear**, **functional**, and **visually appealing** way.  
Feel free to use it as inspiration or integrate it directly into your own dashboard.
